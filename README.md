# 🏷️ QBCore Dynamic Admin Tag (qb-tag)

סקריפט חכם, חינמי, אופטימלי ומתקדם לשרתי FiveM המבוססים על קיוביקור (QBCore) אשר מציג תג אדמין ברור מעננים מעל ראש השחקן (3D Text) יחד עם מרקר (עיגול זוהר) מתחת לרגליים. 

הסקריפט תומך לחלוטין במספר דרגות צוות במקביל (God, Admin, Mod) כך שלכל דרגה ישנו צבע משלה וטקסט שונה אשר מתאימים תצוגה להרשאותיו המדויקות של השחקן (מבוסס על QBCore Permissions או Ace Native).

---

## ✨ פיצ'רים ויתרונות מרכזיים
- 🇮🇱 **תמיכה מלאה בטקסט עברי מובנה (Standalone):** הסקריפט מגיע עם קובץ פונט עברי ייעודי (`out.gfx`) מובנה בתוך תיקיית ה-`stream` של הסקריפט. **אין שום צורך בסקריפט נוסף או עזר חיצוני לשפה העברית!**
- 🚀 **אופטימיזציית שיא (0.00ms במנוחה):** מסונכרן כולו באמצעות **State Bags** של FiveM, בלי לשלוח בקשות שרת-לקוח מיותרות וכבדות. כאשר אין שום אדמין עם התג דולק בסביבת השחקן, קוד ציור הכתב (Tick) נרדם למצב חסכון במשאבים (Sleep Mode).
- 🎨 **עיצוב תלת ממדי ייחודי:** הטקסט שופר והוספו לו הצללה חזקה (Drop Shadow) ומתאר (Outline). כעת הוא נקי, יפה וקריא לחלוטין גם אם עומדים מול השמש.
- ⚙️ **דרגות שונות לפי מערך:** ניתן להוסיף עד אינסוף דרגות צוות ב-Config, כל אחת והצבע / המשפט / המרקר הייחודיים לה.
- 🛠️ **גיבוי הרשאות דינמי חכם:** במקרים בהם מסד הנתונים נופל, הסקריפט ניגש ישירות לבדיקת Ace (יכולות מנהל במערכת ההפעלה של השרת) כך שהצוות תמיד יוכל להדליק את התג על אף תקלות גישה ב-DB.

## 📥 הורדה והתקנה לשרת הקיוביקור שלך

1. הורידו את הקבצים מאתר ה-GitHub ישירות אל תיקיית ה-resources שלכם.
2. ודאו ששם התיקייה שחולצה הוא בדיוק `qb-tag` על מנת למנוע בעיות טעינה (רצוי לשמור את זה בנתיב מסודר כמו בתוך `[qb]`).
3. היכנסו לקובץ ההגדרות של השרת שלכם `server.cfg` והוסיפו את שורת ההפעלה הבאה:
   ```bash
   ensure qb-tag
   ```

## 💻 שימוש ופקודות

להדלקה ולכיבוי התג במשחק עצמו, האדמין צריך רק לרשום בצ'אט:
```text
/tag
```
*(למפתחים, הסקריפט כולל גם פקודת מפתחים ייחודית – `/testtag [rank]` שמאפשרת לבדוק ויזואלית בקלות כל אחת מהדרגות בלי באמת להגדיר גישות מחדש במסד הנתונים!).*

---

## 🛠️ עריכת הדרגות (config.lua)
תוכלו לערוך בקלות כל דרגה, צבע או טקסט בקובץ `config.lua`:

```lua
Config.Tags = {
    ['god'] = {
        Text = "~r~[ הלנהה ]~w~\nשנעי עירפיש ןקחש - טיס ןימדא",
        MarkerColor = { r = 255, g = 0, b = 0, a = 120 }
    },
    ['admin'] = {
        Text = "~b~[ תווצ ]~w~\nשנעי עירפיש ןקחש - טיס ןימדא",
        MarkerColor = { r = 0, g = 150, b = 255, a = 120 }
    }
}
```

> **💡 טיפ חשוב לכתיבה בעברית:**  
> במנוע המשחק של FiveM, פונטים מיוחדים (כמו זה המובנה כאן) מוצגים משמאל לימין. פירוש הדבר הוא שכל משפט עברי בקובץ ההגדרות צריך להקליד **הפוך**.
> למשל, המילה "הנהלה" במקום להיות רשומה כ-`הנהלה`, אמורה להיות רשומה כ-`הלנהה` על מנת שתוצג נכון במשחק.
